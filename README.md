# 🎂 Happy Birthday Sampriti

A beautiful, psychology-themed birthday wish website built with React.

## ✨ Features
- Aurora / cosmic midnight blue theme
- Animated starfield and aurora backgrounds
- Floating particles
- Photo gallery with lightbox
- Psychology profile cards
- Animated message carousel
- Belated birthday wish letter
- Confetti burst interaction

## 🚀 Deploy to GitHub Pages

### Step 1 — Create a GitHub repo
Create a new public repository on GitHub, e.g. `sampriti-birthday`.

### Step 2 — Add your repo URL to package.json
Open `package.json` and add this line (replace with your GitHub username):
```json
"homepage": "https://YOUR_USERNAME.github.io/sampriti-birthday"
```

### Step 3 — Install dependencies
```bash
npm install
```

### Step 4 — Deploy
```bash
npm run deploy
```

This will build and push to the `gh-pages` branch automatically.

### Step 5 — Enable GitHub Pages
In your repo → Settings → Pages → Source → select `gh-pages` branch.

Your site will be live at:
`https://YOUR_USERNAME.github.io/sampriti-birthday`

## 🛠️ Local Development
```bash
npm start
```
